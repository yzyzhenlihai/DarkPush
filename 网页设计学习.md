# 页面中的基本要素：

1.  Logo 网站的标志
2. Navigation 网站的导航
3. Banner 网站的横幅
4. slide 幻灯片
5. Copyright 版权
6. header、footer 头部和尾部

# 浏览器的内核

1. Trident： IE/Edge
2. Gecko：Firefox
3. Webkit：safari /chrome
4. Presto:   opera 

# 域名

.com   企业

.net  从事Internet相关的公司

.org   某个组织

.cn  代表中国自己的互联网标识

.gov.cn 中国政府相关部门

.edu  教育相关
